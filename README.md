# Java Exercises

Set of exercise to learn and practice beginner to intermediate programming topics.

You will be putting into use:

1. Variables
2. Constants
3. Functions and methods
2. Conditional statements
2. Object-Oriented Programming pillars
   3. Abstraction
   4. Encapsulation
   5. Inheritance
   6. Polymorphism

## Getting started

Visit the guides [here](https://gist.github.com/chrisvasqm/a6fca6521b61ddfb39da48ccbe156fe8) and then revisit the code 
on each package/folder to compare your implementation.
